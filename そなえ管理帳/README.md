# そなえ管理帳

防災備蓄と非常持ち出し袋の準備状況を、一目でわかる「そなえ度」としてまとめて管理するツールです。

## どんな機能があるか

- 世帯人数・備蓄日数（3日分／7日分）を入力すると、水・主食・電池などの推奨備蓄量を自動計算
- 各品目の現在の保有量・消費期限（交換目安日）を入力すると、達成率と期限が近い品目の警告（1ヶ月以内は赤、3ヶ月以内は黄）を表示
- 非常持ち出し袋の固定チェックリスト（懐中電灯・モバイルバッテリー・救急セット・現金・常備薬など全16項目）
- 好きな品目を自由に追加登録できるカスタム品目欄
- 備蓄品の達成率と持ち出し袋のチェック率から、全体の「そなえ度」をパーセンテージでダッシュボード表示

## 誰のためのアプリか

一人暮らしの大学生など、防災意識はあっても備蓄品の管理が続かない人のためのアプリです。

## いつ・どんな場面で使うか

- 新生活を始めたタイミングでの備蓄品の見直し
- 防災の日・台風シーズン前などの定期チェック
- 消費期限が近い備蓄品を入れ替えるタイミングの確認

## 使い方

1. `index.html` をブラウザで開く
2. 上部で世帯人数と備蓄日数（3日分／7日分）を設定する
3. 「備蓄品」タブで、持っている品目の保有量と消費期限を入力する（推奨量は自動計算される）
4. 「非常持ち出し袋」タブで、揃っている持ち出し品にチェックを入れる
5. 上部のダッシュボードで「そなえ度」と期限が近い品目の警告を確認する

※ 入力内容はページを開いている間のみ保持されます（ブラウザストレージは使用していないため、閉じると入力はリセットされます）。
