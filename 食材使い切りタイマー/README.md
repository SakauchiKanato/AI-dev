# 食材使い切りタイマー

一人暮らしの大学生が、買った食材の期限を忘れてフードロスを出してしまうのを防ぐための管理ツールです。

## どんな機能があるか

- 食材名・カテゴリ（野菜／肉／魚介／乳製品／主食／その他）・購入日・期限日を登録
- 登録した食材を期限が近い順に一覧表示し、残り日数に応じて警告色（赤＝期限切れ〜1日以内、黄＝2〜3日）を表示
- カテゴリで一覧を絞り込み
- 期限が近い食材について、カテゴリ別の使い切りレシピ案を自動で提案
- 「使った」「捨てた」ボタンで記録し、フードロス削減率（使った数 ÷ (使った数＋捨てた数)）をダッシュボードに表示
- 登録中の食材一覧をワンクリックでクリップボードにコピー

## 誰のためのアプリか

一人暮らしで自炊をする大学生など、買った食材を使い切れずに捨ててしまいがちな人向けです。

## いつ・どんな場面で使うか

買い物から帰ったタイミングで食材を登録し、料理をするときや冷蔵庫を整理するときに一覧をチェックして、期限が近いものから優先的に使うために使います。

## 使い方

1. 食材名・カテゴリ・購入日・期限日を入力して「追加する」を押す
2. 一覧で期限が近い食材（赤・黄で表示）を確認し、下部の使い切り提案を参考にする
3. 使い終えたら「使った」、捨ててしまったら「捨てた」を押して記録する

※ ブラウザの保存機能は使っていないため、ページを閉じると登録内容はリセットされます。必要な内容は「一覧をコピー」でメモしておいてください。
