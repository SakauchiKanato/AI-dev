# サブスク整理カルテ

動画・音楽・AI・ゲームなど複数のサブスクリプションサービスの支払いを一覧で管理し、使っていないサービスを見つけて固定費を見直すためのツールです。

## どんな機能があるか

- サービス名・月額・カテゴリ（動画／音楽／AI／ゲーム／その他）・契約日・使用頻度を登録
- 月額合計と年間換算額を自動計算
- カテゴリ別の支出内訳をグラフで表示
- 使用頻度が「ほぼ使ってない」サービスを金額が大きい順に「見直し候補」として自動抽出
- 「解約した」ボタンで解約済みリストに移動し、削減できた月額・年額を表示
- 一覧をワンクリックでクリップボードにコピー

## 誰のためのアプリか

複数のサブスクに入っていて、合計でいくら払っているか把握しきれていない一人暮らしの人向けです。

## いつ・どんな場面で使うか

物価上昇や円安で固定費を見直したくなったとき、月に一度の家計チェックのタイミングなどに使います。

## 使い方

1. 左のフォームにサービス名・月額・カテゴリ・契約日・使用頻度を入力して「追加する」を押す
2. 登録したサービスは一覧に表示され、右側のダッシュボードに月額合計・カテゴリ別内訳が反映される
3. 「見直し候補」パネルに出てきたサービスを確認し、解約する場合は一覧の「解約した」ボタンを押す

ブラウザで `index.html` を開くだけで使えます。インターネット接続やアカウント登録は不要です。
